Click-through prototypes let you add interactions to elements. You can configure what triggers the interaction and what action it performs.

## Adding interactions

<Frame>
  <img src="/images/prototype-mode/interactions-inspector.png" alt="The Interactions section in the inspector panel showing trigger and action configuration" />
</Frame>

1. Select an element
2. In the inspector, click **Interactions** > <Icon icon="plus" size={16} />
3. Choose a trigger and action

## Previewing interactions

Preview your interactions in [preview mode](/learn/design-mode/preview). For more advanced features like form handling, state management, and dynamic interactions, use [Prototype with AI](/learn/prototype-mode/overview).

## Available prototyping actions

Every prototyping action has a trigger — click, hover, or right-click — that defines what user action starts the interaction.

**On click**
- **Navigate to** — navigates to another page
- **Open overlay** — opens a page as an overlay (dialog or drawer)
- **Close overlay** — closes the current overlay
- **Open dropdown** — opens a dropdown menu
- **Open popover** — opens a popover

**On hover**
- **Open tooltip** — opens a tooltip
- **Open popover** — opens a popover on hover

**On right click**
- **Open context menu** — opens a context menu
