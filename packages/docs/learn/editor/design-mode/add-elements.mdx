title: Add elements
description: Insert components, primitives, and layouts into your designs.
---

Add elements to your canvas using the Insert panel, quick insert, keyboard shortcuts, or the command palette. Subframe provides primitives (Text, Image, Icon, Stack, Divider) and components from your library.

## Insert panel

The Insert panel in the left sidebar shows all available elements organized by category.

**Open the panel** - Click the **Insert** tab in the left sidebar (or press <kbd>A</kbd>).

**Browse elements** - Scroll through categories:

- **Primitives** - Text, Image, Icon, Stack, Input, Textarea, Divider, Loader
- **Components** - Built-in Subframe components and your custom components
- **Snippets** - Pre-built patterns and sections
- **Popovers** - Dropdown, Context menu, Tooltip, Dialog, Drawer

**Add to canvas** - Drag any element from the panel onto the canvas. Drop it into a Stack or other container to nest it.

**Search** - Use the search box at the top to filter elements by name.

## Quick insert

Quick insert lets you add elements inline without opening the Insert panel.

**Trigger quick insert:**

- Press <kbd>Ctrl</kbd> + <kbd>I</kbd> to insert above the selected element
- Press <kbd>Ctrl</kbd> + <kbd>K</kbd> to insert below
- Press <kbd>Ctrl</kbd> + <kbd>J</kbd> to insert left
- Press <kbd>Ctrl</kbd> + <kbd>L</kbd> to insert right
- Click the **+** button that appears when hovering near selected elements

**Choose an element** - A search popup appears at the insertion point. Type to search or browse the list, then click to insert.

**Ask AI to insert** - In the search popup, describe what you want (e.g., "Add a sign up button") and AI creates it for you.

## Primitives

Primitives are the basic building blocks for your designs.

### Stack

Stacks arrange child elements using flexbox. They're the primary layout tool in Subframe.

**Add a Stack** - Insert a Stack from the panel or quick insert.

**Configure layout** - Use the Inspector to set:

- **Direction** - Vertical, horizontal, or wrap
- **Gap** - Spacing between children
- **Alignment** - How children align on main and cross axes
- **Padding** - Internal spacing

Stacks adapt to content automatically. Set child width/height to "hug" to shrink to content, or "fill" to expand.

### Text

Text elements create headings, paragraphs, and labels.

**Add text** - Insert a Text element, then double-click to edit content.

**Style text** - Configure typography in the Inspector:

- Font family and weight
- Size and line height
- Color and alignment
- Text wrap and truncation

Text exports as semantic HTML based on your styling choices.

### Icon

Icons display SVG icons from supported icon libraries.

**Add an icon** - Insert an Icon element.

**Choose icon** - Click the icon picker in the Inspector and search for an icon by name.

**Supported libraries** - Feather Icons, Lucide, Font Awesome, Heroicons, and custom uploaded icons.

**Configure** - Adjust icon size and color in the Inspector.

### Image

Images display pictures from URLs or uploaded files.

**Add an image** - Insert an Image element.

**Set image source** - In the Inspector:

- Paste an image URL
- Or upload a file from your computer

**Configure display** - Set the scale mode:

- **Cover** - Fill the container, cropping if needed
- **Fit** - Show entire image, adding letterboxing if needed

**Accessibility** - Add alt text for screen readers in the Inspector.

### Divider

Dividers create horizontal or vertical lines to separate content.

**Add a divider** - Insert a Divider element.

**Configure** - Set orientation (horizontal/vertical) and color in the Inspector.

## Components

Components are reusable UI elements from your design system.

**Add a component** - Drag it from the Insert panel or use quick insert.

**Component instances** - When you add a component, you create an instance. Instances inherit properties from the component definition.

**Configure instances** - Use the Inspector to override props, text, and styling for each instance.

For more on components, see [Using Components](/components/using-components).

## Command palette

Press <kbd>Cmd</kbd> + <kbd>K</kbd> to open the command palette, then search for elements to insert.

The command palette shows the same elements as the Insert panel but with faster keyboard access.

## Copy and paste

**Copy** - Select an element and press <kbd>Cmd</kbd> + <kbd>C</kbd>.

**Paste** - Press <kbd>Cmd</kbd> + <kbd>V</kbd> to paste into the selected container or at the same level as the selected element.

**Duplicate** - Press <kbd>Cmd</kbd> + <kbd>D</kbd> to instantly duplicate the selected element.

## Positioning elements

When you add an element, Subframe positions it based on:

- Where you dropped it (if dragging)
- The insertion direction (if using quick insert)
- The parent container's layout

Drag elements to reposition them. Blue highlights show valid drop locations.

## Element hierarchy

Elements nest in a parent-child hierarchy. Parents control layout, and children inherit properties.

**View hierarchy** - Use the **Page** tab in the left sidebar (Layers panel) to see the tree structure.

**Navigate hierarchy:**

- Press <kbd>Esc</kbd> to select the parent
- Press <kbd>Tab</kbd> to cycle through siblings
- Click elements in the Layers panel to select them
