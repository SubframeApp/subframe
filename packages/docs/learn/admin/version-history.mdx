## Open version history

1. Click the project switcher
2. Select **Show version history**
3. Select the version timestamp from the dropdown in the banner at the top

## Restore a version

Click **Restore** to revert your project to a previous state. 

When you restore a version, all changes are restored across your project, including:

- Page designs
- Component edits
- Theme changes
- All other project modifications

## Version retention

Version history retention varies by plan:

- **Free plan**: 24-hour history
- **Pro plan**: 7-day history
- **Custom plans**: Longer version history available

Contact the Subframe team for custom version history retention options.
