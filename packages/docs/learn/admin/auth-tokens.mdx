title: Auth tokens

---

Auth tokens let team members authenticate the Subframe CLI with their accounts.

## Create a token

You can create a personal token for yourself at [https://app.subframe.com/cli/auth](https://app.subframe.com/cli/auth).

1. Go to **Settings > Access Tokens**
2. Click **New**
3. Copy the token from the dialog

Tokens are shown once. Store them securely.

<Note>Only Admins and Editors can create tokens. Viewers don't have CLI access.</Note>
