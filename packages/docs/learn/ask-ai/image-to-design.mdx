---
title: Image to design
description: Use images and website URLs as references when generating designs with AI.
---

Upload screenshots or mockups, or paste a link to a live website, and Ask AI recreates designs using your components and theme.

## Upload an image

1. Click the **image icon** in the Ask AI toolbar or drag an image directly into the prompt area.
2. Add a prompt describing how to use the image
3. Press <kbd>Enter</kbd> to generate

AI analyzes the image and creates 1-4 variations that match the structure using your Subframe components.

Supported formats: PNG, JPG, JPEG, WebP

## Import from Figma

1. In Figma, select a frame to copy as PNG (right-click or press <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>C</kbd>)
2. In Ask AI, paste the image into the prompt area
3. Add a prompt and press <kbd>Enter</kbd>

## Reference a website

Paste a link into your prompt and Ask AI captures the page to use as a reference.

1. Paste a URL into the prompt area
2. Add a prompt describing what to use from the page
3. Press <kbd>Enter</kbd> to generate

Ask AI captures the page's layout and styling, then generates variations using your components and theme.

<Note>Ask AI only fetches public URLs that you paste directly into the prompt.</Note>
