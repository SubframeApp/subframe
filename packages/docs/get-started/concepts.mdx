## Projects

Your Subframe project contains your theme, components, and pages. Each project is an independent design system—the source of truth for your product UI.

<img src="https://developers.elementor.com/docs/assets/img/elementor-placeholder-image.png" alt="The Project Switcher dropdown showing a list of projects with a New project button" />

## Components

Reusable UI elements that sync from Subframe to your codebase. The components you design are the same components engineers use in production—design once, use everywhere.

<video autoPlay muted loop playsInline className="w-full" src="https://hevpkratkeuc60w7.public.blob.vercel-storage.com/view-component-documentation-ygcEi4y47JotWdboZUunSHbkxdpYrh.webm" />

[Learn about components →](/components/overview)

## Pages

Individual screens you design with components. Pages export as React code you copy and paste—add your business logic locally in your codebase.

<img src="https://developers.elementor.com/docs/assets/img/elementor-placeholder-image.png" alt="The editor showing a page being designed with components on the canvas" />

## Theme

Design tokens for colors, typography, spacing, shadows, and borders. Your theme syncs as Tailwind configuration, ensuring consistent styling across your design system.

<img src="https://developers.elementor.com/docs/assets/img/elementor-placeholder-image.png" alt="The theme panel showing color tokens, typography settings, and spacing scale" />

[Customize your theme →](/theme/overview)

## Syncing

One-way sync from Subframe to your codebase. Components sync via CLI—changes in Subframe overwrite generated files. All code lives locally for further development.

<img src="https://developers.elementor.com/docs/assets/img/elementor-placeholder-image.png" alt="Terminal showing the sync command running and components being updated" />

[How syncing works →](/developing/syncing-components)

## Ask AI

Generate designs from text prompts or images. AI learns from your designs to create on-brand pages using your theme and components—everything stays editable.

<img src="https://developers.elementor.com/docs/assets/img/elementor-placeholder-image.png" alt="The Ask AI panel with a prompt and multiple design variations generated" />

[Use Ask AI →](/editor/ask-ai/prompt-to-design)

## Prototypes

Interactive prototypes with working navigation and logic. Describe behaviors in natural language—AI generates interactions without writing code.

<img src="https://developers.elementor.com/docs/assets/img/elementor-placeholder-image.png" alt="Prototype mode showing a page with annotations and AI chat panel for adding interactions" />

[Build prototypes →](/editor/prototype-mode/overview)

## MCP server

Connect AI coding tools like Claude Code and Cursor to your Subframe projects. AI assistants fetch components and pages directly—no manual copying needed.

<img src="https://developers.elementor.com/docs/assets/img/elementor-placeholder-image.png" alt="Configuration file showing MCP server setup with Subframe access token" />

[Set up MCP →](/developing/mcp-server)
