## Who should use Subframe?

Subframe bridges the gap between design and development, serving multiple roles on product teams.

**Designers** can create high-fidelity designs without leaving their workflow. The visual editor feels familiar if you've used Figma or other design tools, but what you build is production-ready React code—no handoff required.

**Developers** can skip the tedious work of translating designs into code. Your components sync automatically from Subframe to your codebase, so you can focus on business logic and complex features instead of pixel-pushing.

**Product builders** moving fast can go from idea to working prototype in minutes. Use AI to generate designs from text or images, then export them directly to AI coding tools like Cursor, Bolt, or Replit to build the full application.

## How does it work?

Subframe uses a visual editor to design UI components and pages, then exports them as React + Tailwind code that integrates directly into your codebase.

When you design in Subframe, you're working with real components—not static mockups. These components sync one-way from Subframe to your codebase via the CLI. Change your component design in Subframe, run `sync`, and your code updates automatically.

For pages, Subframe generates the complete React code that you can copy and paste into your project. From there, you add business logic, connect APIs, and build out the functionality.

Your theme (colors, typography, spacing) is defined as Tailwind tokens that sync to your codebase, so your entire design system stays in sync.

## Key features

### Visual design editor

Build interfaces with a drag-and-drop editor that uses flexbox layouts, responsive design, and component composition. Everything you create exports to clean React + Tailwind code.

![The Subframe editor interface showing the visual canvas in the center with a component library on the left and property inspector on the right. A user is dragging a button component onto the canvas.]

### AI-powered design

Generate designs from text descriptions, convert images to working UI, or use inline AI to edit elements on the canvas. Subframe AI understands your brand and can create variations of existing pages instantly.

![The Ask AI panel in Subframe showing a text prompt "Create a pricing page with three tiers" and the generated page design appearing on the canvas with professional styling and layout.]

### Production-ready code export

Export React components that use Tailwind CSS for styling and Radix UI for accessibility. All code is TypeScript, fully typed, and works with any React framework (Next.js, Vite, Astro, Remix).

![Code Mode in the Subframe editor displaying clean TypeScript React code on the left and the rendered component preview on the right, with an "Export code" button highlighted.]

### Component system

Use pre-built components from the library or create custom ones. Components support props, variants, slots, and states—all the patterns you'd expect from a mature design system.

![The Components panel showing a library of pre-built UI components including buttons, inputs, cards, and navigation elements, each with multiple variants displayed.]

### Real-time collaboration

Work with your team in real-time. Multiple people can edit the same project simultaneously, with changes syncing instantly across all collaborators.

![Two user avatars visible in the top-right corner of the editor with cursor indicators showing both users editing different parts of the same page design.]
