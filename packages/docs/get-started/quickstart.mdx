This guide walks you through creating your first design in Subframe and exporting it to code. You'll go from zero to a working page in under 10 minutes.

## What you'll build

By the end of this guide, you'll have:
- A Subframe account and project
- A designed page using AI or templates
- The page exported as React + Tailwind code
- Everything running in your local development environment

## Create your first design

### Sign up and create a project

1. Go to [app.subframe.com](https://app.subframe.com) and sign up
2. After signing in, you'll automatically get a default project
3. Click **New page** to start designing

Your project comes with a complete component library—buttons, inputs, cards, navigation, and more. You'll use these to build your page.

<img src="https://developers.elementor.com/docs/assets/img/elementor-placeholder-image.png" alt="The Subframe dashboard showing a new project with the default component library visible in the left sidebar and a blank canvas in the center" />

### Design with AI

The fastest way to get started is using Ask AI to generate a design:

1. Click **Ask AI** in the top-right corner (or press <kbd>Cmd</kbd> + <kbd>K</kbd>)
2. Describe what you want to build:

> "Create a landing page hero section with a headline about building UIs faster, a subtitle, an email input, and a 'Get Started' button"

3. Click **Generate** and wait a few seconds
4. Choose your favorite variation and click **Apply**

<img src="https://developers.elementor.com/docs/assets/img/elementor-placeholder-image.png" alt="The Ask AI panel showing a text prompt and four generated design variations for a landing page hero section with different layouts and styles" />

The AI generates real, editable components using your theme. You can now switch to Design Mode to make adjustments.

### Or start from a template

Prefer to start with a pre-made design?

1. Click the **+** button in the Pages panel
2. Browse the template gallery
3. Select a template that's close to what you need
4. Customize it in Design Mode

Templates are fully customizable—change colors, text, spacing, and layout to match your vision.

### Make it yours

Switch to **Design Mode** to edit your page:

- **Add components** from the left sidebar by dragging them onto the canvas
- **Edit text** by double-clicking any text element
- **Adjust spacing** by selecting a Stack and changing the gap
- **Change colors** by selecting an element and using the color picker
- **Add your logo** by dragging an image element onto the canvas

Everything is built with real components that will export as production-ready code.

<img src="https://developers.elementor.com/docs/assets/img/elementor-placeholder-image.png" alt="The Subframe editor in Design Mode showing a hero section being edited, with the Inspector panel open on the right displaying color, spacing, and typography controls" />

## Export to code

Now let's get your design into code.

### View the code

1. Click **Code** in the top-right to open Code Mode
2. You'll see the complete React + Tailwind code for your page
3. The code includes all imports, TypeScript types, and styling

<img src="https://developers.elementor.com/docs/assets/img/elementor-placeholder-image.png" alt="Code Mode showing TypeScript React code on the left with proper syntax highlighting and a live preview of the page on the right" />

### Set up your project

Before you can use the code, you need to install Subframe in your project. If you don't have a project yet, create one:

<CodeGroup>
  ```bash npm
  npx @subframe/cli@latest init
  ```
  ```bash yarn
  yarn dlx @subframe/cli@latest init
  ```
  ```bash pnpm
  pnpx @subframe/cli@latest init
  ```
  ```bash bun
  bunx @subframe/cli@latest init
  ```
</CodeGroup>

The CLI will walk you through setup:
1. Choose your framework (Next.js, Vite, or Astro)
2. Authenticate with your access token from [app.subframe.com/cli/auth](https://app.subframe.com/cli/auth)
3. Configure where to sync components (default: `./src/ui`)
4. Let it sync your components and configure Tailwind

This takes about 2 minutes. Once it's done, you'll have all your Subframe components in your codebase.

### Copy and use your code

Back in Code Mode:

1. Click **Copy code**
2. Create a new file in your project (e.g., `src/pages/Hero.tsx`)
3. Paste the code
4. Import it in your app:

```tsx src/App.tsx
import Hero from './pages/Hero'

export default function App() {
  return <Hero />
}
```

Start your dev server and you'll see your design running as real code.

### Add interactivity

The exported code is a static UI. Add your business logic:

```tsx src/pages/Hero.tsx
'use client'
import { useState } from 'react'
import { Button } from '@/ui/Button'
import { TextField } from '@/ui/TextField'

export default function Hero() {
  const [email, setEmail] = useState('')

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault()
    // Add your signup logic here
    console.log('Signing up:', email)
  }

  return (
    <form onSubmit={handleSubmit}>
      {/* Your Subframe design */}
      <TextField.Input
        value={email}
        onChange={(e) => setEmail(e.target.value)}
        placeholder="Enter your email"
      />
      <Button type="submit">Get Started</Button>
    </form>
  )
}
```

For more details, see [Adding business logic](/develop/adding-business-logic).
