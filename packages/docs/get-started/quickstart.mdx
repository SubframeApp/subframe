## Create your project

1. Sign up at [subframe.com](https://subframe.com) to create your first project.
2. Configure your theme with quick presets or customize later in the Theme page.
3. Select a default navigation style for your app.


## Design your first page

You can create a new page with AI, select a template, or start from scratch.

<Steps>
  <Step title="Prompt with AI">
    Ask AI generates multiple design variations using your theme and components.

    1. Go to https://app.subframe.com/design/new to open the new page dialog.
    2. Type a prompt for the page you want to design.
    3. Submit your prompt to open the page editor in Ask AI mode and wait for designs to generate.
    4. Select a variation to preview. Click **Apply design** to add your chosen design to the current page.
    
    <Tip>Mix and match elements from different variations by dragging to the page from the preview.</Tip>
  </Step>

  <Step title="Edit in Design Mode">
    Switch to Design Mode to refine your page in the responsive page editor.

    - Drag-and-drop elements on the page to rearrange.
    - Select an element and click the **+** Insert buttons to add new elements.
    - Edit styles and properties in the Inspector panel on the right.
    - Right-click or press <kbd>/</kbd> for contextual edits to a selected element.
  </Step>
</Steps>

## Set up your codebase

Next, we'll create a new project using the Subframe Vite starter kit to implement your design.


### Build with Subframe MCP


<Steps>
  <Step title="Install the MCP server">
    Switch to **Code** > **Installation** to view Cursor or Claude Code instructions specific to your project.

    Go to https://app.subframe.com/cli/auth to generate your auth token.

  </Step>

  <Step title="Create a new project">
    Open a new folder and copy the installation prompt in **Code** > **Installation** for Cursor or Claude Code.
  </Step>

  <Step title="Prompt to implement your design">
    Switch to **Code** > **Inspect** and copy the MCP link from Subframe to prompt your AI:

    > "Implement the design from this Subframe page [YOUR_PAGE_URL]"

    AI fetches the code via MCP and adds business logic.
  </Step>

  <Step title="Start your development server">
    ```bash
    npm install
    npm run dev
    ```

    Open `localhost:3000`—your page renders exactly as designed.
  </Step>
</Steps>

### Or manually copy code

<Steps>
  <Step title="Create a new Vite project with CLI">
      ```bash npm
      npx @subframe/cli@latest init --dir ./src/ui --alias '@/ui/*' --install --tailwind --sync --template vite
      ```
  </Step>

  <Step title="Copy your page code">
    1. Switch to **Code** > **Inspect** to view React & Tailwind CSS code for your page.
    2. Click the copy icon in the code viewer.
    3. Paste into `src/pages/App.tsx`
  </Step>

  <Step title="Start development">
    ```bash
    npm install
    npm run dev
    ```

    Open `localhost:3000`—your page renders exactly as designed.
  </Step>
</Steps>
