## Introduction

Interactive prototypes enable designers to connect multiple pages within Subframe to test design workflows. Click interactions can be added to any design element.

All your pages are part of a single flow and you can switch between them in order using keyboard shortcuts.

## Adding Click Interactions

**Steps to implement:**

1. Select an element in the editor
2. Click **Prototyping** in the inspector panel
3. Select the destination page for navigation on click
4. Open **Preview** to test the prototype functionality
