## Overview

Subframe's command action menu, activated via <kbd>CMD</kbd> + <kbd>K</kbd>, provides the control you need in context in a searchable interface for design workflows.

## Navigation

Press <kbd>CMD</kbd> + <kbd>K</kbd> from any page to quickly navigate within Subframe:

**Product areas:**
- Templates
- Components
- Theme
- Installation
- Quickstart

**Pages & Components:**
- Create new page
- Create new component

**Settings:**
- Team members
- Projects
- Billing management

## Editor Controls

### Page-level settings

When no layers are selected, you can access:

- Show/hide panels
- Preview page
- View page code
- Apply responsive breakpoints
- Set page layout

### Selected layer properties

When an element is selected, access via <kbd>CMD</kbd> + <kbd>K</kbd> or right-click to modify:

**Actions:**
- Add/remove stack
- Show/hide
- Create component
- Copy code snippet
- Duplicate
- Delete
- Add/remove component slots

**Properties:**
- Quick styles
- Size
- Layout
- Alignment
- Padding
- Text styles
- Text formatting
- Text color
- Background color
- Border
- Corner radius
- Shadows
- Overflow
- Popovers

**Additional options:**
- Insert
- Selection
- Rewrite content with AI
