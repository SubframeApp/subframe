## Overview

You can import your Figma designs into Subframe, perfect for prototyping or creating a design system from scratch.

<Note>
This feature is in a limited beta and not available for everyone. If you're interested in early access, reach out to support@subframe.com
</Note>

## Availability

This import capability requires a custom plan subscription with Subframe.
