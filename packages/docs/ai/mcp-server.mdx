---
title: MCP Servers
---

Subframe provides two MCP servers to enhance your AI workflows:

1. **Subframe**: Access your Subframe components, pages, and themes directly from AI tools
2. **Subframe Docs**: Search docs.subframe.com to help AI tools understand how to use Subframe

<Note>
  Our Subframe MCP server is currently in beta. We'd love your feedback as we continue to improve the features and
  functionality.
</Note>

## Available tools

Our Subframe MCP server provides the following tools:

- **list_components**: List all components in your project
- **list_pages**: List all pages in your project
- **get_component_info**: Generate code for a specific component
- **get_page_info**: Generate code for a specific page
- **get_theme**: Get your project's theme in Tailwind format

Our Subframe Docs MCP server provides the following tools:

- **SearchSubframeDocs**: Searches Subframe's docs page

## Installation

<Tabs>
  <Tab title="Cursor">
    <Steps>
      <Step title="Open MCP settings">
        1. Use `Command` + `Shift` + `P` (`Ctrl` + `Shift` + `P` on Windows) to open the command palette.
        2. Search for "Open MCP settings".
        3. Select **Add custom MCP**. This will open the `mcp.json` file.
      </Step>
      <Step title="Add Subframe MCP servers">
        In mcp.json, add:

```json mcp.json
{
  "mcpServers": {
    "subframe": {
      "url": "https://mcp.subframe.com/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_TOKEN"
      }
    },
    "subframe-docs": {
      "url": "https://docs.subframe.com/mcp"
    }
  }
}
```

Be sure to replace `YOUR_API_TOKEN` with your actual API token. You can create a new API token [here](https://app.subframe.com/settings/tokens).

      </Step>
      <Step title="Test the connection">
        Ask "What tools do you have available?" in Cursor chat to verify both MCP servers are correctly installed.
      </Step>
    </Steps>

    See [Cursor's documentation](https://docs.cursor.com/en/context/mcp#installing-mcp-servers) for more information on installing MCP servers.

  </Tab>

  <Tab title="Claude Code">
    <Steps>
      <Step title="Add the Subframe Docs MCP server">

```bash
claude mcp add --transport http subframe-docs https://docs.subframe.com/mcp
```

      </Step>
      <Step title="Add the Subframe MCP server">

```bash
claude mcp add --transport http subframe https://mcp.subframe.com/mcp --header "Authorization: Bearer YOUR_API_TOKEN"
```

      </Step>
    </Steps>

Be sure to replace `YOUR_API_TOKEN` with your actual API token. You can create a new API token [here](https://app.subframe.com/settings/tokens).

  </Tab>

  <Tab title="Gemini CLI">

    <Steps>
      <Step title="Add Subframe MCP servers">

Add the following to `~/.gemini/settings.json`

```json ~/.gemini/settings.json
{
  "mcpServers": {
    "subframe": {
      "httpUrl": "https://mcp.subframe.com/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_TOKEN"
      }
    },
    "subframe-docs": {
      "httpUrl": "https://docs.subframe.com/mcp"
    }
  }
}
```

      </Step>
    </Steps>

Be sure to replace `YOUR_API_TOKEN` with your actual API token. You can create a new API token [here](https://app.subframe.com/settings/tokens).

See [Gemini's documentation](https://github.com/google-gemini/gemini-cli/blob/main/docs/tools/mcp-server.md#mcp-servers-with-the-gemini-cli) for more information on installing MCP servers.

  </Tab>
</Tabs>
