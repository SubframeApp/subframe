After [installing Subframe](/installation) your theme will be automatically updated in your codebase. The CLI will create a `tailwind.config.js` with your theme tokens.

If you update your theme in Subframe in the future, you can update it in your codebase by [syncing your components](/developing/syncing-components) or manually copy / pasting the `tailwind.config.js` file for your project.

<Note>
  Find the exact tailwind.config.js for your project
  [here](https://app.subframe.com/library?component=theme&showThemeModalCSSType=tailwindV3).
</Note>

<Tab title="Tailwind v4">
Subframe has experimental support for Tailwind CSS v4 in the Subframe CLI. You can [install Subframe](/installation) with the css-type flag:

<CodeGroup>

```bash npm
npx @subframe/cli@latest init --css-type tailwind-v4
```

```bash yarn
yarn dlx @subframe/cli@latest init --css-type tailwind-v4
```

```bash pnpm
pnpx @subframe/cli@latest init --css-type tailwind-v4
```

```bash bun
bunx @subframe/cli@latest init --css-type tailwind-v4
```

</CodeGroup>

After installation completes you will need to import the `theme.css` file into your global css file that is imported into the root of your app - typically `index.css`, `styles.css`, or `globals.css`. Please refer to the [Tailwind CSS v4 docs](https://tailwindcss.com/docs/theme#sharing-across-projects) for more information on importing the `theme.css` file.

```css index.css
@import "tailwindcss";
@import "../ui/theme.css";
```

If you update your theme in Subframe in the future, you can update it in your codebase by [syncing your components](/developing/syncing-components) or manually copy / pasting the `theme.css` file for your project.

<Note>
  Find the exact theme.css for your project
  [here](https://app.subframe.com/library?component=theme&showThemeModalCSSType=tailwindV4).
</Note>

## Dark mode

Subframe does not currently have a built-in way to theme a project for both light and dark mode, but you can implement dark mode in your codebase using CSS variables. See the [Dark mode](/developing/dark-mode) guide to learn how to set up dark mode in your project.
