---
title: Managing projects
---

Projects contain their own theme, components, and pages. All team members can access every project.

## Create a project

1. Click the project switcher in the top nav
2. Click **New project...**
3. Enter a name
4. Choose **Blank project** or **Existing project** to duplicate from another
5. Click **Create**

New projects include a default theme and one blank page.

<Note>
Free plans allow 1 project. Upgrade to Pro for unlimited projects.
</Note>

## Switch projects

Click the project switcher in the top-left to see all projects. Type to search if you have many.

## Rename a project

1. Go to **Settings > Projects** or click **Manage projects** in the project switcher
2. Hover over the project and click the pencil icon
3. Enter the new name and click **Save**

## Duplicate a project

Hover over a project in **Settings > Projects** and click the copy icon. This creates a new project with the same components, theme, and pages.

## Delete a project

Hover over a project in **Settings > Projects** and click the trash icon.

<Warning>
Deleting is permanent. All pages, components, and theme data are erased.
</Warning>

## Sync design system across projects

<Note>
This feature is only available on custom plans. Reach out to **sales@subframe.com** to learn more.
</Note>

If you maintain a separate design system project:

1. Open the project switcher
2. Click **Sync project...**
3. Follow the prompts to merge updates

## Permissions

Only Admins and Editors can create, rename, duplicate, and delete projects.
