## Accessing version history

Click your project name and select **Show version history** to open the timeline view.

<img src="/images/version-history/project-menu.png" alt="Project menu with Show version history option" />

## Project-level versioning

Version history captures everything at the project level. When you restore a version, all changes are restored together, including:

- Page designs
- Component edits
- Theme changes
- All other project modifications

## Timeline view

New versions are automatically saved when you make changes. Navigate through your project history minute by minute using the timeline interface.

<img src="/images/version-history/version-picker.png" alt="Version history timeline showing saved versions" />

Click **Viewing latest version** to browse through previous versions, then click **Restore version** to go back to any point in time.

## Version retention

Version history retention varies by plan:

- **Free plan**: 24-hour history
- **Pro plan**: 7-day history
- **Custom plans**: Longer version history available

Contact the Subframe team for custom version history retention options.
