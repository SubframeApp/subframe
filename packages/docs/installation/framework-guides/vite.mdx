## Installation

Run the following command in the root of your repository to install Subframe and configure your project:

<CodeGroup>
  ```bash npm
  npx @subframe/cli@latest init
  ```
  ```bash yarn
  yarn dlx @subframe/cli@latest init
  ```
  ```bash pnpm
  pnpx @subframe/cli@latest init
  ```
  ```bash bun
  bunx --bun @subframe/cli@latest init
  ```
</CodeGroup>

## Configure import aliases

After installation, you need to configure Vite to resolve import aliases properly.

### Update tsconfig.app.json

Configure the `compilerOptions` in your `tsconfig.app.json` file so TypeScript understands your import aliases:

```json tsconfig.app.json {3-8}
{
  "compilerOptions": {
    "baseUrl": "./",
    "paths": {
      "@/*": [
        "./src/*"
      ]
    },
    "tsBuildInfoFile": "./node_modules/.tmp/tsconfig.app.tsbuildinfo",
    "target": "ES2020",
    "useDefineForClassFields": true,
    "lib": ["ES2020", "DOM", "DOM.Iterable"],
    "module": "ESNext",
    "skipLibCheck": true,

    /* Bundler mode */
    "moduleResolution": "bundler",
    "allowImportingTsExtensions": true,
    "isolatedModules": true,
    "moduleDetection": "force",
    "noEmit": true,
    "jsx": "react-jsx",

    /* Linting */
    "strict": true,
    "noUnusedLocals": true,
    "noUnusedParameters": true,
    "noFallthroughCasesInSwitch": true,
    "noUncheckedSideEffectImports": true
  },
  "include": ["src"]
}
```

### Update vite.config.ts

Install the Node types package and update your Vite config to resolve paths:

```bash
npm install -D @types/node
```

Then update `vite.config.ts`:

```tsx vite.config.ts {7-11}
import { defineConfig } from "vite"
import react from "@vitejs/plugin-react"
import { resolve } from "node:path"

export default defineConfig({
  plugins: [react()],
  resolve: {
    alias: {
      "@": resolve(__dirname, "./src"),
    }
  }
})
```

## Using Subframe components

Now you can import and use Subframe components in your Vite app:

```tsx src/App.tsx
import { Button } from "@/ui/Button";

export default function App() {
  return <Button>Click me</Button>;
}
```
