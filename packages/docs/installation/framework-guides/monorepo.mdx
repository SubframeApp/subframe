Subframe works with Turborepo, Nx, and workspace managers (pnpm, Yarn, npm). Install per-app or create a shared UI package.

## Installation strategies

### Shared UI package (recommended)

Create a shared package for Subframe components. Apps import from this packageâ€”single source of truth, sync once, use everywhere.

### Per-app installation

Install Subframe independently in each app. Simpler setup but duplicates components. Use when apps need different design systems.

## Shared UI package setup

### 1. Create the shared package

For Turborepo, generate a workspace package:

```bash
npx turbo generate workspace --name @repo/subframe --type package
```

Select your TypeScript and ESLint configs as devDependencies, then create config files that extend your shared configurations.

For other monorepo setups, create the package manually:

```bash
mkdir -p packages/ui
cd packages/ui
npm init -y
```

### 2. Initialize Subframe

Run the CLI from your UI package directory:

```bash
cd packages/subframe
npx @subframe/cli@latest init --sync --dir ui
```

<Warning>
Set the import alias to `@repo/subframe/*` (or your package name) during setup. If missed, edit `.subframe/sync.json` and reinitialize.
</Warning>

### 3. Export components

Configure `package.json` to expose components and Tailwind config:

```json packages/subframe/package.json
{
  "name": "@repo/subframe",
  "version": "0.0.0",
  "type": "module",
  "exports": {
    "./ui/index.ts": "./ui/index.ts",
    "./ui/components/*.tsx": "./ui/components/*.tsx",
    "./ui/layouts/*.tsx": "./ui/layouts/*.tsx",
    "./ui/tailwind-config": "./ui/tailwind.config.js"
  },
  "dependencies": {
    "@subframe/core": "latest",
    "react": "^18.0.0"
  },
  "peerDependencies": {
    "react": "^18.0.0"
  }
}
```

### 4. Install in apps

Add the package to your app dependencies:

```json apps/web/package.json
{
  "dependencies": {
    "@repo/subframe": "*"
  }
}
```

Import components:

```tsx apps/web/src/App.tsx
import { Button } from "@repo/subframe/ui/components/Button"

export function App() {
  return <Button>Submit</Button>
}
```

### 5. Configure Tailwind in apps

<Tabs>
  <Tab title="Tailwind v3">
    Import Subframe's preset and include component paths:

    ```js apps/web/tailwind.config.js
    module.exports = {
      presets: [require("@repo/subframe/ui/tailwind-config")],
      content: [
        "./src/**/*.{js,jsx,ts,tsx}",
        "../../packages/subframe/ui/**/*.{js,jsx,ts,tsx}",
      ],
    }
    ```
  </Tab>
  <Tab title="Tailwind v4">
    Add CSS imports and source directive:

    ```css apps/web/src/app/globals.css
    @import "tailwindcss";
    @import "@repo/subframe/ui/theme.css";

    @source "../../packages/subframe/ui";
    ```
  </Tab>
</Tabs>

<Note>
All Subframe-generated imports automatically resolve to the local package. Components sync with `npx @subframe/cli sync --all` from the package directory.
</Note>
