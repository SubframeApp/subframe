## Getting Started quickly with `@subframe/cli`

If you already have repository set up that you want to set up your Subframe project in, you can just run the following command in your project directory:

<CodeGroup>
  ```bash npm
  npx @subframe/cli@latest init
  ```
  ```bash yarn
  yarn @subframe/cli@latest init
  ```
  ```bash pnpm
  pnpm @subframe/cli@latest init
  ```
  ```bash bun
  bunx @subframe/cli@latest init
  ```
</CodeGroup>

We currently offer templates for NextJS, Vite and Astro.
